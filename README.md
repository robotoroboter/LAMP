# LAMP
This is my implementation of the LAMP stack, as a Docker Compose. It is made to help me understand LAMP and Docker.

For the LAMP's elements, I've used those:
  * L - AlmaLinux,
  * A - Apache,
  * M - MySQL,
  * P - PHP,
as Docker Compose's services.
